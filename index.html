teste 1 - git clone (Comando responsável por clonar o projeto para minha maquina) 2 - git status (Mostra os status das alterações, vermelho para novos arquivos ou alterados e verde para arquivos adicionados) 3 - git add nome_do_arquivo ou "." ou uma pasta ex contato/ (Adiciona o arquivo ou pasta para enviar ao Github) 4 - git commit -m "aqui entra uma mensagem referente a alteração" (Marca uma versão com os comentários das alterações) 5 - git push (Envia as alterações e commits para o Github) 6 - git pull (Atualiza a versão dos arquivos na maquina local)
